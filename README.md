# rust_2048
An implementation of the game 2048, written in Rust. Use WASD in order to specify move direction.

```
+----+----+----+----+
|    |   2|   4|   4|
+----+----+----+----+
|    |  16|  32|   2|
+----+----+----+----+
|    |   2|   8|   4|
+----+----+----+----+
|    |    |    |    |
+----+----+----+----+

Use WASD to specify merge direction. Press `q` to quit.
```
